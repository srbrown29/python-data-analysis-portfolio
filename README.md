# Python Data Analysis Portfolio

This repository contains selected Python data analysis and machine learning projects demonstrating practical analytical workflows, statistical modelling, and data visualisation techniques.

The projects focus on applying reproducible data-science methods to real-world style datasets using Python and common machine-learning libraries.

---

## Projects

### Customer Segmentation with Clustering

This project explores customer segmentation using unsupervised machine learning techniques including K-Means clustering, PCA, and t-SNE visualisation.

**Skills demonstrated:**

* Exploratory Data Analysis (EDA)
* Feature engineering
* Data preprocessing
* Unsupervised machine learning
* Cluster evaluation
* Data visualisation

**Technical stack:**
Python | pandas | NumPy | scikit-learn | matplotlib | seaborn | PCA | t-SNE | K-Means

---

### Anomaly Detection in Ship Engine Sensor Data

This project investigates anomaly detection techniques using multivariate engineering sensor data. Statistical and machine-learning approaches are used to identify abnormal operational behaviour.

**Skills demonstrated:**

* High-dimensional data analysis
* Outlier detection
* PCA
* Isolation Forest
* One-Class SVM
* Statistical analysis
* Data visualisation

**Technical stack:**
Python | pandas | NumPy | scikit-learn | matplotlib | seaborn | PCA | Isolation Forest | One-Class SVM
