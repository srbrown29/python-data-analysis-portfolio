# Anomaly Detection in Ship Engine Sensor Data

## Overview

This project investigates anomaly detection methods using multivariate ship engine sensor data. Statistical and machine-learning techniques are applied to identify abnormal operational patterns within complex engineering datasets.

---

## Objectives

* Explore high-dimensional sensor data
* Detect statistical outliers
* Apply dimensionality reduction techniques
* Implement anomaly detection models
* Compare model outputs
* Visualise anomalous behaviour

---

## Technical Stack

Python | pandas | NumPy | scikit-learn | matplotlib | seaborn | PCA | Isolation Forest | One-Class SVM

---

## Methods Used

* Exploratory data analysis
* IQR-based outlier detection
* Feature scaling
* Principal Component Analysis (PCA)
* Isolation Forest
* One-Class SVM
* Data visualisation
