# Customer Segmentation with Clustering

## Overview

This project explores customer segmentation using unsupervised machine learning techniques in Python. The analysis investigates behavioural patterns within customer data to identify meaningful customer groups.

---

## Objectives

* Perform exploratory data analysis
* Prepare and transform features for machine learning
* Apply K-Means clustering
* Evaluate cluster quality
* Visualise clusters using PCA and t-SNE
* Generate interpretable analytical insights

---

## Technical Stack

Python | pandas | NumPy | scikit-learn | matplotlib | seaborn | PCA | t-SNE | K-Means

---

## Methods Used

* Data cleaning and preprocessing
* Feature scaling
* K-Means clustering
* Elbow method
* Silhouette analysis
* Principal Component Analysis (PCA)
* t-SNE visualisation
